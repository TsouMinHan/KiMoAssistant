從資料庫中取得每個任務的執行時間並安排好行程，將執行後的結果儲存入資料庫中，另外儲存通知的訊息內容(用於DiscordBot）。

任務內容有：

1. 爬新聞資訊 - 爬取文章直到有重複的內容
2. 爬巴哈姆特文章 - 過濾重複文章以及GP數量
3. 爬書籍資訊 - 顯示特價書籍資訊
4. PTT文章 - 顯示二手交易以及其他看板文章
5. 發送個人提醒訊息
6. 定期整理資料庫（刪除重複資料）
